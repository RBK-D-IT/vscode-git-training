## 🚀 Quick Start: Setup Instructions

Follow these steps to prepare your workspace on your **AWA Windows VM**:

1. **Download the Repository:**
   * On this page, click the green **Code** button at the top right.
   * Select **Download ZIP**.
2. **Extract the Files:**
   * Extract the downloaded ZIP folder specifically to your **`Downloads`**
     folder (resulting in `Downloads/vscode-git-training/`).
   * *Note: The notebook contains hardcoded directory checks and path commands
     that expect this specific folder structure to run correctly.*
3. **Open in VS Code:**
   * Open VS Code on your VM.
   * Go to **File → Open Folder...** and select the extracted
     `Downloads/vscode-git-training/` folder.
4. **Open the Notebook:**
   * Open the interactive notebook file:
     **`VSCode_Git_Workflow_Training.ipynb`** and follow the instructions to
     begin.
5. **Session-Day Redownload:**
   * While the notebook is near-complete, minor changes may be made prior to
     the session. **Please download and extract a fresh ZIP on the day of the
     workshop** to ensure you are working with the most up-to-date content.
